## Export steps

1. Open `Payments_Optimization_Dashboard.pbix` and refresh data (`powerbi-data/` CSVs).
2. Confirm the first page is named **Executive Summary**.
3. **File → Export → Export as image** for each page (or snip at 1280×720).
4. Save PNGs here with the filenames above.
